## About The Project

TrustAI is an innovative platform focused on bringing transparency, fairness, and explainability to banking AI systems. It empowers banks to meet regulatory requirements while building customer trust through ethical AI governance. The platform provides AI decision explanations, consent management, bias detection, audit logging, and role-based access control.

## Features

- User registration and authentication with role-based access control  
- AI decision explanations with SHAP/LIME integration  
- Dynamic privacy consent management  
- Real-time bias detection and alerts  
- Audit logging for transparency and compliance  
- Responsive banking-grade UI leveraging Bootstrap 5
